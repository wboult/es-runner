ES Runner launches official Elasticsearch ZIP distributions in a local process, waits for readiness, and provides a small Java API for interaction. These docs are organized with the **Diataxis** framework so you can choose the style that fits your goal.

## Choose your path

- **Tutorials**: learn step-by-step with guided walkthroughs.
- **How-to Guides**: solve a specific task (cluster setup, mirrors, plugins, CI).
- **Reference**: look up configuration keys, APIs, and compatibility.
- **Explanation**: understand design decisions, tradeoffs, and policies.
